# Nebula Live OS

## Live Environment

This project is currently running in **Live Environment**.

- **Database**: Connected to **Neon Serverless Postgres**.
- **Persistence**: Channels, Users, and Licenses are stored securely in the cloud.
- **Zero Config**: Just run `npm run dev` and start exploring.

## Architecture
- **Frontend**: React 19, Vite, Tailwind CSS
- **State**: Zustand
- **Database**: Neon Postgres (@neondatabase/serverless)
